# 🌍 Overview

**AeroVision 3D** is a modern interactive aviation learning platform designed to make aircraft engineering concepts engaging, visual, and easy to understand.

Instead of presenting information as static text, AeroVision combines **interactive aircraft systems**, **real-time 3D visualization**, **smooth animations**, and **modern UI/UX** to provide an immersive educational experience.

This project showcases advanced frontend development while helping users explore aviation through interactive technology.

# 🎯 Why I Built This Project

AeroVision 3D was created to bridge the gap between traditional aviation education and modern interactive web experiences.

Learning aircraft systems from textbooks can often feel overwhelming. This project transforms those concepts into an engaging visual experience through interactive diagrams, animations, and 3D visualization.

The goal was not only to create an educational platform but also to challenge myself by learning advanced frontend technologies such as **Three.js**, **GSAP**, and **Lenis**, while building a professional portfolio project inspired by modern product websites.
